# 常用快捷键

```
⌘（command）+D
 选词（反复按快捷键，即可继续向下同时选中下一个相同的文本进行同时编辑）
⌘（command）+P
 搜索项目中的文件
⌘（command）+ CTRL + p
 打开项目列表面板
Ctrl + `
命令控制台
```
```
新建文件: command + n

在另一个sublime text窗口中全屏新建文件: command + shift + n

光标多选: command + 点击

跳转到当前行末尾: command + right

跳转到当前行行首: command + left

快速复制当前行: command + shift + d

按单词跳转: option + left | right

鼠标拖拽垂直性光标多选: option + 鼠标垂直拖拽

选择当前文档内所有相同单词: control + command + g

恢复之前关闭的文件: command + shift + t

新增分割窗口: command + k + up (新分割的窗口默认将当前文件转换过去)

恢复分割窗口: command + k + down

显示/隐藏侧边栏: command + k + b

进入免打扰模式: control + command + shift + f (默认工作永远只编辑一个文件)

连选单词: command + d

向下连选行: command + l

查找文件: command + p | t

查找行: control + g

查找函数: command + r

快速调整文本位置: control + command + up | down

注释/取消注释: command + / (单行)

注释/取消注释: command + option + / (多行)

删除当前行光标之后的所有文字: command + k + k | control + k

删除当前行光标之前的所有文字: command + delete

删除当前行: control + shift + k

选择括号内所有文字: control + shift + m

将光标返回到上次光标所在位置: control + -

多重光标选择: control + shift + up | down

在距离最近的小括号/花括号/方括号前后移动光标: control + m

在当前行前面插入一行: command + shift + enter

在当前行后面插入一行 ： command + enter

打开/关闭python控制台: control + ｀

改变当前单词/选中内容为大写: command + k + u

改变当前单词/选中内容为小写: command + k + l

调出粘贴板历史纪录: command + option + v

选择html标签内的内容: command + shift + a

在标签页之间切换: command + option + left | right

设置/清除mark: command + f2

跳转到上一个mark: shift + f2

跳转到下一个mark: f2

```
